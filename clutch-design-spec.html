Лента — взаимодействия

1
Баланс (тап) → Кошелёк. Показывает доступные USDT.
2
Карточка активной дуэли (тап) → Комната дуэли. Статус: таймер / «ждёт ответа» / «на арбитраже».
＋
Центральная кнопка таб-бара → Создать дуэль.
Лента — это «домой». Сюда возвращаемся после любого действия. Лента активности друзей создаёт виральность и FOMO.

Друзья — взаимодействия

1
«Вызов» или тап по другу → Создать дуэль с этим соперником, уже подставленным.
2
«Пригласить по ссылке» — генерирует инвайт (Telegram/ссылка). Пришедший по ссылке попадает сюда в список. Это главный канал роста.
У каждого друга видно его рекорд и ранг — это подогревает желание «вызвать сильного». Поиск — для больших списков.

Создать дуэль — взаимодействия

1
Соперник. Подставлен из Друзей. Кнопка «+» добавляет ещё игроков → групповая дуэль.
2
Ставка — ручной ввод. Большое поле = пользователь печатает любую сумму. Чипы $10/25/50/100 снизу — быстрая подстановка (не единственный способ).
3
Стороны — редактируемые. Создатель сам задаёт текст обеих сторон (это не только про опоздание: «Барса»/«Реал», «Я прав»/«Ты прав») и выбирает свою. Соперник автоматически получает другую.
4
«Отправить вызов» → списывает твою ставку в эскроу и создаёт Комнату в статусе «ожидание». Сопернику уходит вызов.
ИИ-Судья по тексту условия предлагает чёткую формулировку и критерий победы перед отправкой (анти-«почти успел»).

Вызов получен — взаимодействия

1
«Принять» → списывает ставку соперника в эскроу, дуэль становится активной → оба в Комнате.
2
«Отклонить» → дуэль отменяется, ставка инициатора возвращается из эскроу. Инициатору приходит уведомление.
3
«Изменить условия» → контр-предложение (другая сумма/формулировка). Возвращается инициатору на подтверждение.
Этот экран открывается из пуша/уведомления «Тебе бросили вызов». Деньги соперника блокируются только в момент принятия.

Комната дуэли — взаимодействия

1
ИИ-Судья в чате. Системные сообщения: зафиксировал условие, ведёт отсчёт, по итогу постит вердикт прямо в ленту комнаты.
2
Блок развязки (появляется на дедлайне). «Я выиграл» обеими сторонами совпало → авто-выплата → Вердикт. Если кто-то жмёт «Не согласен» → Арбитраж.
💬
Чат-подколы между участниками — это и есть «комната, где спорят». Удерживает в дуэли и делает её весёлой.
Для групповой дуэли здесь несколько участников и несколько баров. Чат — главный социальный и удерживающий элемент.

Арбитраж — взаимодействия

1
Пруфы сторон. Гео, скрин, фото со временем. У кого пруф сильнее — того и правда. Видно, кто что приложил.
2
«Добавить пруф» — камера/файл. Фото для дисциплины-режима снимаются «в моменте» (анти-фейк), не из галереи.
3
«Получить вердикт» → ИИ выносит прозрачное решение → Вердикт. Несогласие → апелляция на живого арбитра (платно).
Качество и прозрачность Судьи = главный moat. Вердикт всегда с объяснением «почему».

Вердикт — взаимодействия

1
«Реванш» → Создать дуэль с тем же соперником и условием (подогревает H2H-вражду).
2
«Забрать банк» → выигрыш зачислен на баланс в Кошельке (при non-custodial — выплата по смарт-контракту).
📤
Добавить кнопку «Поделиться» — генерит карточку результата для Stories/ТикТока. Это бесплатный канал привлечения.
Экран проигравшего — зеркальный, но с мотивацией: «−15 рейтинга, реванш?» Без унижения, чтобы хотелось отыграться.

Кошелёк — взаимодействия

1
«Пополнить» → пополнение баланса в USDT (адрес для депозита / Telegram-кошелёк на TON). На MVP — крипта.
2
«Вывести» → вывод USDT на кошелёк пользователя. Важно по безопасности: приложение не хранит приватные ключи и реквизиты; ввод адреса делает сам пользователь.
3
История — все операции: ставки, выигрыши, пополнения, выводы, комиссии.
Часть баланса может быть «в эскроу» (заморожена в активных дуэлях) — показываем отдельно, чтобы не путать с доступной суммой.

Профиль — взаимодействия

1
Настройки (⚙) → уведомления, язык, привязка кошелька, выход.
2
Лига друзей (тап) → таблица лидеров и личные счёты H2H («Ты 4–1 Саша»).
📊
Уровень / XP / ранг / стрик / винрейт / титулы — вся прокачка, которую ты хотел.
«Очки Чести» (платит ли достойно, не юлит ли) можно показать отдельной метрикой — это уникальный, некопируемый слой доверия.
Что я добавил сверх запроса (для целостности и интуитивности):
  • Лента активности друзей — виральность и FOMO прямо на главной.
  • Экран «Вызов получен» с принять / отклонить / изменить — без него дуэль не запускается.
  • Чат в комнате дуэли — то самое «место, где друзья спорят», удержание и веселье.
  • Кнопка «Поделиться» на вердикте — карточка результата в Stories/ТикТок как канал роста.
  • Разделение баланса на доступный и «в эскроу», заметка по безопасности (не храним ключи/реквизиты).
  • Зеркальный экран проигрыша с мотивацией на реванш.
Дальше дорабатываем: онбординг и приглашение друзей, пуш-уведомления, групповые дуэли (UI на 3+ участников), состояния загрузки/пустые экраны, и финальный нейминг с перекраской под бренд. Скажи, с чего начать — и какой экран довести до пиксель-перфекта первым.
